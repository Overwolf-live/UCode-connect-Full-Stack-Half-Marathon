<abbr> Description

Tag <abbr> indicates that the character sequence is an abbreviation. Using the title attribute given decoding of the abbreviation, which makes it possible to understand the abbreviation for those people who are not familiar with it. Besides, search engines index the full-text version of the abbreviation, which can be used to boost rating of the document. Browsers do not highlight text in any way inside <abbr>, with the exception of Odivera before version 15, which adds to text dotted underline.

Syntax

<abbr>Текст<abbr>

<cite>Description

Tag <cite> marks the text as a quote or footnote to another material. This selection is convenient for changing styling text via CSS, and is also used to separate HTML code into structural elements. Browsers usually set the text inside the container <cite> in italics.

Syntax

<cite>Текст</cite>

Example

<div>Description

Element <div> is a block element and is intended to select a fragment of a document with the purpose changes in view content. Typically, the appearance of a block is controlled by styles. In order not to describe the style inside every time tag, you can highlight the style to an external style sheet, and add a class or id attribute for the tag with the name selector.As with other block elements, the content of the tag <div> always starts with a new one strings. A line break is also added after it.

Syntax

<div>...</div>

<nav>Description

Tag <nav> sets site navigation. If there are several blocks of links on the page, then in <nav> usually place priority links. It is also acceptable to use multiple tags <nav> in the document. Is prohibited invest <nav> inside <address>.

Syntax
